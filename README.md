# plug.zsh-theme

Clone this repository and add the theme in the path 

```
$ git clone https://github.com/suyog-kulkarni-pst/ZSH_PLUG.git
$ cd plug.zsh-theme
$ make path
```

Go to your .zshrc file and change ZSH_THEME to this new one

```
$ gedit ~/.zshrc
``` or 
```
$ cd ~/.zshrc 
$ ZSH_THEME=plug
```
