# plug.zsh-theme

Clone this repository and add the theme in the path 

```
$ git clone https://github.com/suyog-kulkarni-pst/ZSH_PLUG.git
$ cd plug.zsh-theme
$ make path
```

Go to your .zshrc file

```
$ gedit ~/.zshrc ## for linux users
``` 
or 
```
$ code . ~/.zshrc 
```
and change ZSH_THEME to this new one
```
$ ZSH_THEME="plug"
```
